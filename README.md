## 📊 精選專案 (Featured Projects)
### 1. 顧客體驗問卷回饋分析 (Customer Experience Analysis)
- 核心目標: 透過分析 85,907 筆問卷數據，識別影響顧客滿意度（CSAT）的關鍵因子，並為管理層提供具體的訓練與流程優化策略 。
- 關鍵技術:
-- 統計分析: 描述性統計、相關性分析（R²）與 ANOVA 變異數分析 。
-- 決策架構: 使用「問題、服務、人員、產品」四大面向拆解指標 。
- 核心洞察: 識別出「訂單」類別中的「物流延遲」與「安裝示範」為主要低分痛點，建議建立 5 分鐘內的初次回覆 SLA 。
- 專案連結: 點此查看完整分析細節

- 核心目標: 針對醫療數據進行 ETL 處理與報表自動化。

## 📈 分析方法論 (Methodology)
> 在處理數據專案時，我遵循以下標準流程：
- 問題定義: 與需求端溝通，釐清商業的核心痛點。
- 數據清洗: 使用 SQL, Excel, 或 Python 進行去識別化與 ETL，確保數據品質。
- 洞察轉化: 將統計分析結果轉化為可執行的行動建議 。
